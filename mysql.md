# mySQL

### Commands
* List databases: `SHOW DATABASES;`
* Connect to db: `USE [db name];`
* List tables: `SHOW TABLES;`
* Describe/show columns: `DESCRIBE [table];`
* Exit/quit shell: `\q`

### Select
```sql
# Count number of distinct values
> SELECT count( DISTINCT(column_name) ) FROM table;

```
